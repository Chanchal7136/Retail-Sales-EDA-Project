# Retail-Sales-EDA-Project
Complete Exploratory Data Analysis of a 50K messy retail dataset including cleaning, visualization, business insights, and anomaly detection.
Project Overview

This project performs Exploratory Data Analysis (EDA) on a 50,000-record messy retail sales dataset. The dataset contained missing values, duplicate orders, inconsistent city names, invalid quantities, and pricing anomalies.

The project includes:

Data Cleaning

Data Transformation

Descriptive Analysis

Time-Based Analysis

Customer Behavior Insights

Pricing & Discount Analysis

Geographic Insights

Outlier Detection

Business Recommendations
